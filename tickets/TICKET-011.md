## 工单描述

根据 Langger 产品特点（语言学习、科学记忆、三种学习模式），重新设计首页 UI 和交互，提升用户体验和产品价值感知。

## 详细需求

### 产品特点分析

Langger 的核心特点：

1. **三种学习模式**：闪卡模式、测试模式、复习模式
2. **科学记忆理论**：艾宾浩斯遗忘曲线、SM-2 间隔重复算法
3. **学习数据统计**：每日进度、学习统计、掌握度跟踪
4. **PWA 能力**：离线使用、安装到桌面

### 当前首页问题

1. **信息展示不足**：仅显示简单的学习进度，未充分利用空间
2. **功能入口不明显**：三种学习模式入口不清晰
3. **视觉层次不明确**：缺乏重点突出和引导
4. **数据可视化缺失**：学习统计和进度缺乏可视化展示
5. **产品价值感知弱**：用户难以快速理解产品核心价值

### 设计目标

1. **清晰的功能入口**：三种学习模式入口突出、易于访问
2. **数据可视化**：学习进度、统计数据的可视化展示
3. **个性化推荐**：根据学习情况推荐合适的学习模式
4. **激励和反馈**：学习成就、连续学习天数等激励元素
5. **现代化设计**：符合当前设计趋势，美观且易用

### 设计方案

#### 布局结构

```
┌─────────────────────────────────────┐
│  顶部：欢迎信息 + 学习统计卡片       │
│  - 今日学习目标进度条                │
│  - 连续学习天数                     │
│  - 总学习单词数                     │
├─────────────────────────────────────┤
│  主要功能区：三种学习模式卡片        │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐│
│  │ 闪卡模式 │ │ 测试模式 │ │ 复习模式 ││
│  │ 图标+描述│ │ 图标+描述│ │ 图标+描述││
│  │ 推荐标签 │ │ 推荐标签 │ │ 推荐标签 ││
│  └─────────┘ └─────────┘ └─────────┘│
├─────────────────────────────────────┤
│  复习提醒区域（如有到期复习）        │
│  - 复习通知卡片                     │
│  - 快速进入复习                     │
├─────────────────────────────────────┤
│  学习数据可视化（可选，折叠）       │
│  - 学习趋势图表                     │
│  - 掌握度分布                       │
└─────────────────────────────────────┘
```

#### 详细设计

**1. 顶部统计卡片**

- **今日学习目标**：

  - 进度条显示：已完成/目标数
  - 百分比显示
  - 完成度颜色变化（0-50% 红色，50-80% 黄色，80-100% 绿色）

- **连续学习天数**：

  - 大号数字显示
  - 火焰图标或日历图标
  - 鼓励文案（如"连续学习 7 天，继续保持！"）

- **总学习单词数**：
  - 累计学习单词数
  - 掌握单词数
  - 进度百分比

**2. 学习模式卡片**

每个模式卡片包含：

- **图标**：代表学习模式的图标（闪卡、测试、复习）
- **标题**：模式名称
- **描述**：简要说明模式特点（1-2 句话）
- **推荐标签**：根据学习情况显示推荐标签
  - "推荐"：适合当前学习阶段
  - "新单词"：有新单词需要学习
  - "复习提醒"：有到期复习
- **快速统计**：显示该模式的相关数据
  - 闪卡：今日学习数
  - 测试：今日测试数、正确率
  - 复习：到期复习数

**3. 复习提醒区域**

- 仅在有到期复习时显示
- 卡片样式，突出显示
- 显示：单词集名称、复习阶段、需要复习的单词数
- 快速进入按钮

**4. 学习数据可视化（可选）**

- 默认折叠，点击展开
- 显示：
  - 最近 7 天学习趋势（折线图）
  - 掌握度分布（饼图或柱状图）
  - 学习模式使用统计

### 交互设计

1. **卡片点击**：

   - 点击学习模式卡片 → 进入学习模式选择页面
   - 点击复习提醒卡片 → 直接进入复习模式

2. **动画效果**：

   - 卡片悬停：轻微上移 + 阴影增强
   - 卡片点击：轻微缩放反馈
   - 进度条：平滑动画填充

3. **响应式设计**：
   - 竖屏：单列布局，卡片垂直排列
   - 横屏：多列布局，卡片横向排列
   - 适配不同屏幕尺寸

### 视觉设计

1. **色彩方案**：

   - 主色调：保持现有蓝色主题（#00b4ff）
   - 模式卡片：不同颜色区分（闪卡-蓝色、测试-绿色、复习-紫色）
   - 进度条：根据完成度变色

2. **图标设计**：

   - 使用统一的图标库
   - 图标风格：简洁、现代、易于识别

3. **卡片设计**：
   - 圆角：16px
   - 阴影：轻微阴影，增强层次感
   - 背景：磨砂玻璃效果或纯色背景

### 技术实现要求

1. **组件化设计**：

   - `HomeStatsCard.tsx`：统计卡片组件
   - `StudyModeCard.tsx`：学习模式卡片组件
   - `ReviewNotificationCard.tsx`：复习提醒卡片组件
   - `LearningChart.tsx`：学习数据图表组件

2. **数据获取**：

   - 从 `wordStore` 获取学习数据
   - 从 `reviewStore` 获取复习提醒
   - 从 `dailyStats` 获取每日统计

3. **性能优化**：

   - 图表使用轻量级库（如 `recharts` 或 `chart.js`）
   - 数据懒加载：图表数据按需加载
   - 使用 `React.memo` 优化组件渲染

4. **国际化**：
   - 所有文案使用 i18n
   - 支持中英文切换
