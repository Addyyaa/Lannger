# Langger 语言学习工具

Langger 是一款针对个人词汇学习场景打造的轻量化应用，结合闪卡式复习、学习统计以及 PWA 能力，帮助学习者在桌面和移动端保持统一体验。

## 主要特性

- ✨ **智能调度算法**：根据单词掌握度、正确率与复习间隔动态调整闪卡推送顺序与频率，兼顾效率与记忆保持。
- 📦 **批量导入**：支持多种格式导入词库，适合按教材单元、主题等方式快速搭建学习内容。
- 📊 **学习数据面板**：展示每日目标、已学词汇、连续学习天数等指标，方便了解进度与复习需求。
- 📱 **PWA 支持**：可安装到桌面/主屏幕离线使用，提供安装横幅以及一键清缓存能力。
- 🔄 **版本提示机制**：自动在管理页展示当前版本与构建时间，当检测到新的 Service Worker 时弹出“发现新版本”提醒，可一键刷新。
- 🛠 **自动版本号递增**：执行 `npm run build` 会触发脚本更新 `version.json`、`src/version.ts` 及 Service Worker 缓存名称，确保每次部署都能正确通知用户刷新。

## 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:5173/` 即可开发调试。生产构建命令：

```bash
npm run build
```

构建产物位于 `dist/`，可直接部署至 GitHub Pages 或任意静态托管服务。

## 部署提示

1. 仓库内已集成 GitHub Actions，推送到 `main` 会自动构建并发布到 `gh-pages`。
2. 若自定义托管方式，请一并上传 `404.html`，避免刷新二级路由（如 `/manage`、`/study`）时报 404。
3. 首次安装或发布新版本后，如提示未立即出现，可在管理页点击版本徽章触发缓存清理，再刷新页面。

## 目录概览

- `src/`：React 组件与业务逻辑。
- `public/sw.js`：Service Worker 缓存和更新策略。
- `scripts/bump-version.mjs`：构建前自动递增版本号与缓存名。
- `version.json`：记录最近一次构建的版本与时间。
